# graphqlReactBoilerplate-master !

## Quick start

- Clone the repo: `https://github.com/hwolf0610/graphqlReactBoilerplate-master.git`.

### Server
- Install with `cd server `
- Install with `npm i `

### dashboard
- Install with `cd dashboard `
- Install with `npm i `

### Project Run
- Project run  `npm run dev ` .
